LAHNTAL2026 – WEBSITE VERSION 1.2a

Diese Version enthält:
- das überarbeitete Website-Layout
- die aktualisierte Startseite
- direkte Hotel- und Service-Links
- die NEUE Tourmappe auf Basis deiner zuletzt hochgeladenen PDF
- eine neue Cache-Version, damit nicht mehr die alte PDF angezeigt wird

HOCHLADEN IN GITHUB

1. Repository „Lahntal2026“ öffnen.
2. Add file → Upload files.
3. Diese vier Dateien hochladen:
   - index.html
   - styles.css
   - sw.js
   - Lahntal_Tourmappe_2026.pdf
4. Commit changes anklicken.
5. 1–2 Minuten warten.
6. Website neu laden.
7. Falls noch eine alte Fassung erscheint:
   - Mac: Cmd + Option + R
   - iPhone/iPad: Safari-Tab schließen und neu öffnen
   - notfalls Website-Daten für github.io löschen

WICHTIG
Die PDF-Datei muss im Repository exakt „Lahntal_Tourmappe_2026.pdf“ heißen,
damit der Tourmappe-Button der Website sie korrekt öffnet.
