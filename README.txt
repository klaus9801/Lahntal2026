LAHNTAL 2026 – GITHUB PAGES

1. Entpacke die ZIP-Datei.
2. Öffne dein GitHub-Repository.
3. Lade ALLE Dateien aus dem Ordner in das Repository hoch.
4. In GitHub: Settings → Pages.
5. Unter „Build and deployment“:
   Source: Deploy from a branch
   Branch: main / root
6. Speichern. Nach wenigen Minuten zeigt GitHub die Webadresse an.

Die Datei index.html ist die Startseite.
Die PDF und das Titelbild müssen im gleichen Ordner liegen.

Hinweis:
Die Bahn-Schaltflächen öffnen bewusst die offizielle DB-Startseite. 
Die konkreten Reisedaten werden vorher per „Angaben kopieren“ in die Zwischenablage gelegt.
